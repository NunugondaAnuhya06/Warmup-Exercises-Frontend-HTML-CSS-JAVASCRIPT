Introduction to Formatting Tags

Attributes of Marquee Tags in Formatting Tags

"Marquee Tags consists of 10 types of Atrributes :"

1.bgcolor=".." [used to Specify Background Color]

2.Behavior=".." [used to Specify Behavior of the test Slide/Scroll/Alternate] Default Behavior is Scroll

3.title=".." [used to Specify the tool tip text]

4.Scrollamount=".." [Specifies the speed value]

5.Loop=".." [specifies the number of times]

6.width=".." [Specifies the marquee width]

7.Height=".." [Specifies the marquee Height]

8.Hspace=".." [Specifies the Horizantal space]

9.Vspace=".." [Specifies the Vertical space]

10.Direction=".." [specifies the direction left/right/top/bottom] Default is left
